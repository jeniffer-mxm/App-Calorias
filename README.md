# App de Controle de Calorias 🍎

Um aplicativo web simples para **monitorar a ingestão de calorias**, desenvolvido com **Python (Flask) no backend** e **HTML/CSS/JS no frontend**.  

O app permite que o usuário registre alimentos, visualize seu consumo diário e tenha um front-end interativo.

---

## 🚀 Funcionalidades

- Registrar alimentos consumidos com quantidade e calorias.
- Visualizar o consumo diário.
- Frontend responsivo com formulário de entrada de dados.
- Backend em Python Flask com endpoints para manipulação dos dados.
- Fácil deploy em plataformas gratuitas como Render ou Railway.

---

## 🛠 Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask, Gunicorn  
- **Controle de versão:** Git & GitHub  
- **Hospedagem gratuita sugerida:** Render, Railway  
- **Banco de dados opcional:** SQLite (local), Supabase ou Firebase
